# Zapret GUI

Современное приложение (CustomTkinter) для обхода блокировок **Discord, YouTube
и Telegram** на Windows. Надстройка над DPI-движком `winws.exe` + WinDivert из
открытого проекта [zapret](https://github.com/Flowseal/zapret-discord-youtube).

> ⚙️ Не VPN. Локальный обход DPI: правит сетевые пакеты, чтобы цензор не мог
> опознать и заблокировать соединение.

## Возможности
- **Дашборд на главной** — статус обхода, живое «здоровье» (Discord/YouTube/
  Google + Telegram-прокси) и кнопки Старт/Стоп одной плиткой.
- Запуск/остановка обхода выбранным **пресетом**; служба автозапуска Windows и
  полный автозапуск (приложение + обход + прокси при входе в систему).
- **Умный авто-поиск** стратегии: пресеты пробуются по приоритету (последний
  рабочий → запасной пул → похожие по типу десинка), быстрый режим с ранней
  остановкой, асинхронные проверки хостов.
- **Авто-восстановление** (watchdog): при деградации связи переключает стратегию
  из пула, а при исчерпании пула — сам запускает авто-поиск.
- **Свои сайты** — добавьте домены для обхода (пишутся в `list-general-user.txt`).
- **Автообновление** списков и IP-набора (`ipset-all`) из upstream по расписанию.
- **Диагностика** — проверка окружения (права, BFE, драйвер WinDivert, конфликты,
  DNS, доступность сайтов) со светофором и кнопками авто-починки.
- **Telegram-прокси** — встроенный MTProto-WS-прокси (отдельный exe не нужен):
  запуск/остановка, ссылка `tg://`, статистика соединений, лог, тумблер
  Cloudflare-фолбэка.
- **Шифрованный DNS (DoH)**, исключения Windows Defender, тёмная/светлая тема
  «Signal» + выбор акцента.
- **Журнал** с фильтром и копированием; **самообновление** через GitHub Releases.
- Пресеты как данные (`presets.json`), контроль целостности бинарников (SHA-256).

## Установка
Скачайте `ZapretControl.zip` из [Releases](../../releases/latest), распакуйте
папку `ZapretControl` куда удобно и запустите `ZapretControl.exe` из неё. Всё
нужное (winws, WinDivert, списки, пресеты, TgWsProxy) уже внутри. Права
администратора запрашиваются автоматически.

> Формат **onedir** (exe + папка `_internal`) выбран намеренно: нет распаковки
> во временную папку при каждом запуске — быстрее старт и нет ошибок загрузки
> Python DLL при самообновлении.

> ⚠️ Windows Defender/SmartScreen может ругаться (как и на сам zapret) — это
> ложное срабатывание. См. раздел «Антивирус» ниже.

## Антивирус / SmartScreen
Приложение распаковывает и запускает `winws.exe` + драйвер WinDivert, поэтому
антивирусы иногда дают ложное срабатывание. Без платной подписи кода это не
устранить полностью, но можно убрать на своём ПК:

- **В приложении:** «Управление» → «Настройки» → **«Добавить в исключения»**
  (Defender) — добавляет папку приложения и `winws.exe` в исключения Windows
  Defender (нужны права администратора).
- **SmartScreen** при первом запуске: «Подробнее» → «Выполнить в любом случае».
- Вручную: «Безопасность Windows» → «Защита от вирусов и угроз» →
  «Параметры» → «Исключения» → добавить папку с приложением.

## Сборка из исходников
```bat
pip install customtkinter pyinstaller
build_exe.bat
```
`build_exe.bat` обновляет `presets.json` из `.bat` и собирает `ZapretControl.exe`.

## Обновления / релизы
Приложение проверяет последний релиз (`releases/latest`) и предлагает обновиться
(качает `ZapretControl.zip`, заменяет установку и перезапускается).

Выпуск новой версии автоматизирован: поднять `APP_VERSION` в `zapret_core.py`,
запушить тег `vX.Y.Z` — GitHub Actions прогоняет тесты и линт, и только при их
прохождении собирает и публикует релиз с `ZapretControl.zip`.

## Тесты
Юнит-тесты чистых функций ядра — в `tests/`. Локально:
```bat
pip install pytest ruff
pytest tests/ -q
ruff check --select=E9,F63,F7,F82 .
```
В CI это гейт перед сборкой: падают тесты — релиза не будет.

## Структура
| Файл | Назначение |
|------|------------|
| `zapret_app.pyw` | интерфейс (CustomTkinter) |
| `zapret_core.py` | логика: пути, пресеты, процессы/служба, обновления, проверки |
| `presets.json` | декларативные пресеты (стратегии как данные) |
| `bin/`, `lists/` | движок и списки (из проекта zapret) |
| `ZapretControl.spec`, `build_exe.bat` | сборка |
