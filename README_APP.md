# Zapret Control — современное приложение

Самодостаточное приложение с тёмным интерфейсом (CustomTkinter) для обхода
блокировок **Discord, YouTube и Telegram**.

## Запуск (один файл)

- **`ZapretControl.exe`** — самодостаточный. Внутрь зашиты `winws.exe`, драйвер
  WinDivert, списки, пресеты и `TgWsProxy_windows.exe`. Качать что-либо ещё не нужно.
- При **первом запуске** разворачивает рядом с собой нужные файлы (`bin\`,
  `lists\`, `presets.json`, пресеты, `TgWsProxy_windows.exe`) и сверяет целостность
  бинарников по SHA-256. Положите exe в отдельную папку.
- Python не требуется. Права администратора запрашиваются автоматически.
- Из исходников: `pythonw zapret_app.pyw` (нужны `customtkinter`; `presets.json`
  создаётся автоматически из `.bat`).

> ⚠️ Один файл, разворачивающий и запускающий `winws.exe`, драйвер ядра и прокси,
> может срабатывать у Windows Defender/SmartScreen. «Подробнее → Выполнить в любом
> случае» и/или добавить в исключения.

## Архитектура

| Файл | Назначение |
|------|------------|
| `ZapretControl.exe` | самодостаточное приложение (раздавать только его) |
| `zapret_app.pyw` | интерфейс (CustomTkinter) |
| `zapret_core.py` | логика: пути, пресеты, процессы/служба, проверки, целостность, TgWsProxy |
| `presets.json` | **декларативные пресеты** (стратегии как данные, а не разбор .bat) |
| `ZapretControl.spec`, `build_exe.bat` | сборка exe |
| `utils/app_config.json` | состояние (выбранный пресет, путь к TgWsProxy) |

## Разделы
- **🛡 Управление** — статус, запуск/остановка, выбор пресета, служба автозапуска,
  настройки (игровой фильтр, обновления, IPSet), инструменты.
- **🔍 Авто-поиск** — двухфазный подбор лучшей стратегии (асинхронные проверки).
- **✈ Telegram** — запуск/остановка TgWsProxy.
- **📜 Журнал**.

## Что нового (Фаза 1: независимость от апстрима)
- **Стратегии как данные** — все пресеты вынесены в `presets.json` (id, name,
  label, args с плейсхолдерами), приложение больше не зависит от формата `.bat`
  (они остаются для импорта/совместимости).
- **Контроль целостности** — при запуске `.exe` встроенные бинарники сверяются
  по SHA-256 и восстанавливаются при повреждении.
- **Надёжные пути состояния** — конфиг/флаги в рабочей папке, с фолбэком в
  `%LOCALAPPDATA%\ZapretControl`, если рядом с exe писать нельзя.

## Пересборка
`build_exe.bat` (нужны `pip install customtkinter pyinstaller`). Обновляет
`presets.json` из `.bat` и собирает exe.
